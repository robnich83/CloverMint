# SFT → NFT Ticketing dApp Scaffold

This project demonstrates a decentralized ticketing system where event tickets are minted as ERC-1155 SFTs, and after the event, holders can convert them into ERC-721 collectible NFTs.

## Structure
- `contracts/` – Solidity smart contracts
- `frontend/` – React/Next.js front-end scaffold
- `tests/` – Hardhat test examples

## Usage
1. Install dependencies with `npm install`.
2. Compile and test using `npx hardhat compile` and `npx hardhat test`.
3. Run the local app with `npm run dev`.
